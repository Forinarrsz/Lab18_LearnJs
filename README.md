## Краткое описание работы

Лабораторная работа посвящена сравнительному анализу языков программирования C# и JavaScript. В ходе работы были изучены основные синтаксические и семантические различия между языками, рассмотрены особенности типизации, выполнения кода, работы с данными и реализации объектно‑ориентированного программирования.

## Описание (что изучили)

В рамках лабораторной работы были рассмотрены следующие аспекты:

* **Типизация:** статическая в C# против динамической в JavaScript.
* **Процесс выполнения кода:** компиляция в промежуточный код (IL) в C# и интерпретация/JIT‑компиляция в JavaScript.
* **Работа с переменными и константами:** синтаксис объявления и особенности использования.
* **Вывод данных:** механизмы вывода информации в консоль.
* **Проверка типов:** когда и как выполняется проверка типов данных.
* **Операторы сравнения:** различия в поведении операторов `==` и `===`.
* **Условные конструкции:** синтаксис и возможности `if/else`, `switch` и тернарного оператора.
* **Циклы:** доступные виды циклов и их синтаксис.
* **Функции:** особенности объявления и использования функций/методов.
* **Работа с массивами и объектами:** способы создания и использования структур данных.
* **ООП:** подходы к реализации объектно‑ориентированного программирования (классическое в C# и прототипное в JavaScript).
* **Специальные значения:** работа с `null` и `undefined` в обоих языках.
* **Строгость языка:** влияние компилятора C# и необходимость дисциплины в JavaScript.

## Итоговая таблица: JavaScript vs C#

| Аспект | C# | JavaScript |
|--------|------|------------|
| Типизация | Статическая | Динамическая |
| Компиляция | Да (в IL‑код) | Нет (интерпретация/JIT) |
| Точка входа | `Main()` | Отсутствует |
| Переменные | `int x = 5;` | `let x = 5;` |
| Константы | `const int X = 5;` | `const X = 5;` |
| Вывод в консоль | `Console.WriteLine()` | `console.log()` |
| Проверка типа | Во время компиляции | `typeof` |
| Сравнение | `==` (по значению) | `===` (строгое) |
| Условия | `if/else/switch` | `if/else/switch` + тернарный оператор |
| Циклы | `for/while/do‑while/foreach` | `for/while/do‑while/for...of` |
| Функции | Методы с типами | Функции без типов |
| Массивы | `int[] arr = new int[5];` | `let arr = [];` |
| Объекты | Требуют класс | Можно создать литералом |
| ООП | Классическое | Прототипное |
| Null | `null` | `null` и `undefined` |
| Строгость | Высокая (компилятор) | Низкая (нужна дисциплина) |
